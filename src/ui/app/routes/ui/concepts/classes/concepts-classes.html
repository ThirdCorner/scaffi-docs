This is the base essence of a class. All classes come with a constructor function which gets called when the class is loaded.

In order to use this class outside a file, you add the syntax export default ClassName;. This will allow you to instantiate the class in other parts of the program.

If the class has any decorators, then Angular will be in charge of loading and instantiating for you.

Extends

With the advent of ES6, we can extend classes properly in Javascript. This means that you can declare a base class of functionality for a page, and then make sub pages that extend that base class to tweak your pages in different ways.

For more information, see Class Inheritance
