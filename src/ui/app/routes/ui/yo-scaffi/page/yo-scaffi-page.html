yo scaffi:page

Call this when you want to make a route page.

Usage

yo scaffi:page

In-depth Description

When a ui page is made, Scaffi creates it in the src/ui/app/routes/**/* folder, based on the route you provide it. This will give you an html and js controller. Because of how things are structured, the page will be accessible via url once it's created.

If you say you want to get a split controller between mobile and web, Scaffi will make two controllers, ending in *.mobile.js and *.web.js respectively. This will allow you to specify platform-specific logic.
