yo scaffi:theme

Call this when you want to set the theme for the project.

Usage

yo scaffi:theme

In-depth Description

When you create a project, this generator is actually called before the install process ends. Thus, you can technically call it on a project if you want to switch between bootstrap or material. If you do change the theming engine of an app, you will have to manually replace any components in use in the app and change the includes in the /src/ui/app/theme/theme.js file.
