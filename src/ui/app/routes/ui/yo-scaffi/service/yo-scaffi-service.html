yo scaffi:service

Call this when you want to make an Angular service.

Usage

yo scaffi:service

In-depth Description

When a ui service is made, Scaffi creates it in the src/ui/app/services folder. This is a traditional Angular service which you can inject in your page or component controllers. Scaffi also generates a mock service (for prototype mode if you need to override certain calls) and a fixture file to store structures for prototype and documetation usages.
