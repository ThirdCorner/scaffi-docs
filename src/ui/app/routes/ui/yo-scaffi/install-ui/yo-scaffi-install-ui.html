yo scaffi:install-ui

Call this when you want to install a package for the ui portion of a project. This will be accessible for all mobile and web app builds.

Usage

yo scaffi:install-ui ng-table

yo scaffi:install-ui github:angular-wizard/angular-wizard

In-depth Description

!!! We don't want to use npm install * directly anymore, because there are processes in the framework that keep everything in-sync for not only the build, but for everyone on the project. Always use one of the yo scaffi:install-* commands when adding a package. !!!

This command is used if you want to install npm/github packages that will be used in all platform builds, from mobile to web. Use it like you would npm install.
