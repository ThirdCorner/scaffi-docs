yo scaffi:component

Call this when you want to make an angular component for use in your project.

Usage

yo scaffi:component

In-depth Description

When a ui component is made, it creates it in the src/ui/app/components folder. This will give you an html and js controller. You can treat this as a app-wide stub page.
