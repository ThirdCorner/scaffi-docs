yo scaffi:ionic

Call this when you want to use one of the ionic cli commands on your project. Based on the command, Scaffi will redirect it to the appropriate folder.

Supported Commands

yo scaffi:ionic browser * *

yo scaffi:ionic info

yo scaffi:ionic emulate *

yo scaffi:ionic platform *

yo scaffi:ionic plugin * *

yo scaffi:ionic run *

yo scaffi:ionic resources

In-depth Description

Because Scaffi structure is not exactly Ionic structure, most commands have been brought in to work as if it were ionic.

Some commands are NOT supported, such as serve (which is synonymous to yo scaffi:start *) and build (which is synonymous to yo scaffi:build * *)

Also to note, currently, options do not get passed into an ionic command, such as platform --nosave. Until this issue is fixed, if you need to run an ionic command with an option flag, or a command that is not currently supported, you can navigate to /src/ui/mobile/ and run direct ionic commands there. If you need to run an ionic command that relies on a ui, you can run the yo scaffi:build command and then run your ionic command after navigating to /src/ui/build/[platform]/public.
