yo scaffi:stub

Call this when you want to make a page section that has a lot of logic or template data.

Usage

yo scaffi:stub

In-depth Description

When a ui stub is made, Scaffi creates it in the src/ui/app/routes/**/* folder, depending on what route you provide. A stub acts exactly like a component, but positioned in a route so that you can break up large sections of page. Because it acts like a component, this means you can pass data to it through the scope object in the stub controller.
