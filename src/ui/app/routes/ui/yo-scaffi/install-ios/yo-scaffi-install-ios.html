yo scaffi:install-ios

Call this when you want to install a package for ios mobile builds only.

Usage

yo scaffi:install-ios ng-table

yo scaffi:install-ios github:angular-wizard/angular-wizard

In-depth Description

!!! We don't want to use npm install * directly anymore, because there are processes in the framework that keep everything in-sync for not only the build, but for everyone on the project. Always use one of the yo scaffi:install-* commands when adding a package. !!!

This command is used if you want to install npm/github packages that will only be used for an ios mobile build. Use it like you would npm install.
