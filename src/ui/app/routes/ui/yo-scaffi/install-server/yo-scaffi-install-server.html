yo scaffi:install-server

Call this when you want to install a package for the scaffi server of a project.

Usage

yo scaffi:install-server lodash

yo scaffi:install-server github:npm/node-semver

In-depth Description

!!! We don't want to use npm install * directly anymore, because there are processes in the framework that keep everything in-sync for not only the build, but for everyone on the project. Always use one of the yo scaffi:install-* commands when adding a package. !!!
