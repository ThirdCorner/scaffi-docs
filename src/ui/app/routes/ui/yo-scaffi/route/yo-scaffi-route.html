yo scaffi:route

Call this when you want to make a route.

Usage

yo scaffi:route

In-depth Description

When a ui route is made, Scaffi creates it in the src/ui/app/routes/**/* folder. If you provide the cli a route, it will create the new route in a nested location. This will give you a js ui-router controller, as well as a page index. Because of how things are structured, the page will be accessible via url once it's created.

If you say you want to get a split controller between mobile and web, Scaffi will make two controllers, ending in *.mobile.js and *.web.js respectively. This will allow you to specify platform-specific logic.
