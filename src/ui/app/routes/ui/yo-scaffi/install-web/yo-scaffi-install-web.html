yo scaffi:install-web

Call this when you want to install a package for web builds only. This means that the package will be excluded from all android and ios builds.

Usage

yo scaffi:install-web ng-table

yo scaffi:install-web github:angular-wizard/angular-wizard

In-depth Description

!!! We don't want to use npm install * directly anymore, because there are processes in the framework that keep everything in-sync for not only the build, but for everyone on the project. Always use one of the yo scaffi:install-* commands when adding a package. !!!

This command is used if you want to install npm/github packages that will only be used for web builds. Use it like you would npm install.
