yo scaffi:uninstall-server

Call this when you want to uninstall a server package.

Usage

yo scaffi:uninstall-server *

In-depth Description

!!! We don't want to use npm uninstall * directly anymore, because there are processes in the framework that keep everything in-sync for not only the build, but for everyone on the project. Always use one of the yo scaffi:uninstall-* commands when adding a package. !!!

We want to call this instead of npm uninstall * because it updates the build process so it doesn't get included. It also updates the package.json, so it's a good cleanup command to run. Plus you can call this command anywhere in the project.
