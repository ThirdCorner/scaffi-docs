yo scaffi:build

Call this when you want to build an app into a certain platform and environment.

Usage

yo scaffi:build [platform] [environment]

Platforms: web, ios, android

Environment: prototype, localhost, development, ci, qa, production

In-depth Description

As a broad overview, when this command is run, it bundles up any necessary app resources, and runs browserify on the app.js to generate the app script bundle. If it's building for mobile, it moves everything to the builds/ios|android folder. If it's building for web, it moves the front-end resources to the src/server/public folder, and then moves the entire server folder to builds/web/server folder.
