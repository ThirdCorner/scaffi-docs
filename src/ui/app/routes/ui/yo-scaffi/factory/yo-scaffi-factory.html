yo scaffi:factory

Call this when you want to make an angular factory for use in your project.

Usage

yo scaffi:factory

In-depth Description

When a ui factory is made, Scaffi creates it in the src/ui/app/factories folder. This will create a js controller that is used like a typical angular factory.

Let's say we generate a factory called 'LaunchToast'.
