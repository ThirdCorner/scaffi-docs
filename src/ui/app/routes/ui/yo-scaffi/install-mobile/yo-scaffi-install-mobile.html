yo scaffi:install-mobile

Call this when you want to install a package for android and ios mobile builds only.

Usage

yo scaffi:install-mobile ng-table

yo scaffi:install-mobile github:angular-wizard/angular-wizard

In-depth Description

!!! We don't want to use npm install * directly anymore, because there are processes in the framework that keep everything in-sync for not only the build, but for everyone on the project. Always use one of the yo scaffi:install-* commands when adding a package. !!!

This command is used if you want to install npm/github packages that will only be used for mobile builds. Use it like you would npm install.

This is how we manage to inject ionic into mobile builds, while excluding it from web builds. In src/ui/app/theme/theme.js we do this:

The import 'ionic-angular/release/js/ionic.js'; will be imported into the vendor.js file for any mobile build, while being excluded from web builds.
