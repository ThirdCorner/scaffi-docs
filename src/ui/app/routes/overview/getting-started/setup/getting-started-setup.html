Disabling Cache

You need cache disabled so that when you make changes to your code, the browser doesn't keep an old compiled build of your code. When you have Chrome's console open, you can tell it not to cache.

  1. With chrome focused, hit F12 to open up the Developer Tools window.
  2. Click on the Network Tab.
  3. Make sure the "Disable cache" checkbox is checked.
