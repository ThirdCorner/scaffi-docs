Installing Packages

You’ll usually need to leverage other packages for an app so you’re not reinventing the wheel. In this instance, we want an idle checking component.

Kill your Scaffi start instance and run the following:

yo scaffi:install-ui ng-idle

What this does is run an npm install command for you and ensures the package.json file gets updated. Not only that, but it updates the build system so that if there are other coders on the project, this will make sure they get the package installed when they run yo scaffi:start web.

If we wanted to install a server package, the command would be:

yo scaffi:install-server [package]
