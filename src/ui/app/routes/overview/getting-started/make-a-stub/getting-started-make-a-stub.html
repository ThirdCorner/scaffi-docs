Make A Stub

Now that our app is hooked up to a database, let’s do a bit of cleanup to get used to some of the other process changes in Scaffi. We have some duplicated code in the products list page, so let’s clean it up and componentize it. In products-index.html, we’re duplicating the display for active and completed items. Since there’s a good chunk of code there, let’s make it a reusable stub.

A stub is the same as a component, only a stub is used for a specific page. You can make a stub if you have repeating display elements or if you want to break up a template page that’s more than 200 lines.

If you have the server running, go ahead and kill it and run:

yo scaffi:stub

  • Stub Name: list
  • Route: products/index
  • Hint enter for basic
  • Separate Controllers: n

This will add a products-index-list.html and a products-index-list.stub.js to your routes/products/index folder.
