We’re assuming you have Node.js >= 4.0 and NPM installed. Make sure the following are also installed:

  • npm install -g yo
  • npm install -g generator-scaffi
  • npm install -g gulp
  • For Scaffi UI
  • npm install -g jspm
  • For Scaffi Server
  • npm install -g nodemon

These all need to be globally installed, otherwise you’ll run into issues.

Open up a CMD window, navigate to your projects folder and run the following:

yo scaffi

Answer the questions and it’ll generate a scaffold project for you and install any npm dependencies.

Once everything’s installed, you’re ready to start the app.
